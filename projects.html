Tanlagan loyihalarim

Portfolio saytim

Shaxsiy portfolio saytim HTML, CSS va JavaScript yordamida ishlab chiqilgan. Saytda men haqimda, blog, loyihalar va aloqa bo‘limlari mavjud.

Notepad dasturi (Adalo)

Adalo platformasi orqali foydalanuvchi login, ro‘yxatdan o‘tish, matn yozish va saqlash funksiyalari mavjud bo‘lgan oddiy Notepad ilovasi yaratdim.

To-do List App (JavaScript)

Soddalashgan vazifa ro‘yxati ilovasi. Foydalanuvchi vazifalarni qo‘shish, belgilash va o‘chirish imkoniyatiga ega.

Weather App (API bilan)

JavaScript yordamida ochiq ob-havo API orqali hozirgi ob-havoni ko‘rsatadigan veb-ilova.

Texno Blog (Markdown/JSON)

Yangi texnologiyalar, dasturlash tillari haqida maqolalar yoziladigan oddiy blog sahifasi yaratdim.

Telegram Bot (Python)

Foydalanuvchi bilan interaktiv muloqot qiladigan Python asosida yaratilgan oddiy Telegram bot.

Responsive galereya (Modal oynali)

Loyihalar rasmlari ko‘rsatiladigan interaktiv galereya modal oynalar bilan. Mobil qurilmalar uchun ham moslashgan.
